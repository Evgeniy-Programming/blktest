Простой скрипт на Python для тестирования производительности блочного устройства с помощью fio и создания графика с помощью gnuplot.


использование:
blk test -name=<имя_теста> -имя файла=<путь к_файлу> -выходные данные=<путь к_отходу_png>
blktest -name=<test_name> -имя файла=<path_to_file> -выходные данные=<path_to_output_png>

<test_name>: имя теста.
<path_to_file>: путь к файлу блочного устройства (пример: /dev/sda).
<path_to_output_png>: путь к сгенерированному файлу png. -->
